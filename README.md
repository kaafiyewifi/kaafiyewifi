# Kaafiye WiFi

Hotspot Management System + Router Integration (MikroTik RouterOS v7) + WireGuard + RADIUS.

## Stack
- Laravel 11 (PHP 8.3+)
- MySQL
- Tailwind CSS v4 (Blade)
- MikroTik RouterOS v7 + WireGuard + RADIUS

## Key Features (WIP)
- Multi-location management
- Router register + heartbeat API
- Script generator (one script per hotspot)
- Captive portal templates
